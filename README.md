## Getting Started

If you'd like to copy the portfolio, see below:

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/djordan218/portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
